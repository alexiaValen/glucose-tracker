# Glucose Tracker

A modern glucose tracking dashboard built with React + TypeScript.

## ✨ Features
- Add glucose entries (value + timestamp)
- View history of entries
- Basic trend visualization
- Clean, minimal UI

## 🛠 Tech Stack
- React
- TypeScript
- Tailwind CSS
- Recharts

## 🚀 Getting Started

```bash
npm install
npm run dev
