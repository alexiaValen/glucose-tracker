# GraceFlow Icon Usage Guidelines

## When to Use Each Icon

### SignalRingThin
- Loading states
- Passive connectivity
- Background sync indicator

### SignalRingActive  
- Active sync in progress
- Data being transmitted
- Connection established

### AxisMarker
- Timeline position indicators
- Progress through a flow
- Cycle phase markers

### VeinDivider
- Section separators
- Content breaks
- Group dividers

### InsightUnderline
- Intelligent insight text
- AI-generated content
- Data-backed statements

### SeverityContinuum
- Symptom severity display
- Pain level indicators
- Any 1-10 scale

### EmptyStateSprout
- No data yet states
- First-time user flows
- "Getting started" screens

### ChipIndicatorTick
- Multi-select UI
- Checkbox alternatives
- Selection confirmation

## Size Guidelines

- **16px**: Inline with small text, tight spaces
- **20px**: Inline with body text
- **24px**: Standard UI element (default)
- **32px**: Section headers, emphasis
- **48px**: Hero icons, empty states

## Color Usage

- **Ink (#2B2B2B)**: Primary content, active states
- **Muted (#CFC9BF)**: Secondary content, dividers
- **Accent (#B89A5A)**: Emphasis, active indicators
